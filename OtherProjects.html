The Baby Saver, Spring 2018

This project was developed using an arduino and buttons to act as pressure sensor The goal of the project was to create a tool to be used by paretns to alert them if they ahve left thier child in the back seat of a car. The device would use blue tooth disconnecting (~15') to alert the parent so they could then go back to the car to grab thier child. This project was done during a hackathon and was praised by the President of Florida Polytechnic University for the goal and purpose of the project
