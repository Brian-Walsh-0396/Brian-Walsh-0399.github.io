The Baby Saver, Spring 2019

This project was developed using an arduino and buttons to act as pressure sensor

The goal of the project was to create a tool to be used by paretns to alert them if they ahve left thier child in the back seat of a car. The device would use blue tooth disconnecting (~15') to alert the parent so they could then go back to the car to grab thier child. This project was done during a hackathon and was given distintion by the Dean of Florida Polytechnic University
